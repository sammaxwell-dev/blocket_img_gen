# PRD-lite: SnapturaAI MVP

## Проблема
Продавцы на Blocket/Tradera фотографируют товары дома → фон захламлён, свет плохой → мало просмотров → товар не продаётся.

**Статистика**: качественные фото = +30–94% конверсии в просмотры/покупки.

## Решение
Веб-сервис, который за <15 сек превращает "домашнее" фото в "готовое к публикации":
- Удаление/замена фона на нейтральный
- Автокоррекция освещения и цвета
- Smart-кроп 1:1 (1200×1200) под мобильный просмотр маркетплейсов

## Целевая аудитория

| Сегмент | Описание | Объём |
|---------|----------|-------|
| **Primary** | Физлица, 2–10 товаров/мес | 80% рынка |
| Secondary | Профи-продавцы, 50+ товаров | v1.1 |

**Портрет пользователя**: продаёт детские вещи, мебель б/у, электронику. Фотографирует на телефон дома. Хочет быстро выложить и продать.

## Ценностное предложение
> "Загрузи сырое фото → скачай готовое к продаже за 10 секунд"

## Функции MVP

### Core (Must Have)
| Функция | Описание |
|---------|----------|
| Загрузка фото | Drag&drop (desktop) + camera/gallery (mobile), до 10 фото |
| Удаление фона | AI-ремувал через Remove.bg API |
| Цветокоррекция | Автоматическая коррекция света/баланса (Replicate GFPGAN) |
| Smart Crop | Обрезка 1:1 с автофокусом на объект (Cloudinary) |
| Предпросмотр | До/После toggle в карусели |
| Скачивание | ZIP-архив готовых фото |
| Лимиты | 5 фото/день бесплатно, 99 SEK/мес — unlimited |

### Out of Scope (v1.1+)
- Лейблы (pris/storlek/skick)  
- История/галерея в ЛК
- API для профи
- Множественные кропы (4:5, 16:9)
- Годовая подписка
